# Screenplay CLI

## About

The official CLI for Screenplay. To run, `screenplay install <xcode_project>` (also, simply running `screenplay` will show help).

## Developing

1. `yarn install`
2. `yarn build`
3. `yarn link`

You should now be able to call `screenplay` from the command line.
